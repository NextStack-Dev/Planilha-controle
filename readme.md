# Controle de Calibração dos Equipamentos

Projeto simples para controle de vencimento de calibração de equipamentos.

## Visão geral

Pequena aplicação estática (HTML/CSS/JS) que exibe uma lista de equipamentos
e suas datas de vencimento para controle de calibração.

## Estrutura do projeto

- **index.html** — página principal com a tabela de equipamentos.
- **css/style.css** — estilos da página.
- **js/jscript.js** — script principal (funcionalidades do front-end).
- **js/script.js** — scripts adicionais (se aplicável).
- **LICENSE** — licença do projeto.

## Uso

1. Abra o arquivo `index.html` no navegador.
2. A tabela exibirá os equipamentos e suas colunas: Equipamento, Modelo, Empresa, Vencimento.

## Como editar

- Para alterar o layout ou estilos, edite [css/style.css](css/style.css).
- Para alterar a lógica ou adicionar dados, edite [js/jscript.js](js/jscript.js) ou [js/script.js](js/script.js).
- Para alterar o conteúdo estático, edite [index.html](index.html).
